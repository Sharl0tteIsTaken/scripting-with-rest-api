# Scripting-with-REST-API
This repository contains scripts that use Python to interact with the GitHub REST API.

Script location and description:
 - [Create issues](https://github.com/Sharl0tteIsTaken/Scripting-with-REST-API/blob/main/scripts/create_issue.py): Create issues based on the provided template, and replace the placeholders with the replacements in [placeholder.json](https://github.com/Sharl0tteIsTaken/Scripting-with-REST-API/blob/main/assetsplaceholder-create_issue.json). Assets used for this script are from the `first contribution` repository, please refer to issue [#107627](https://github.com/firstcontributions/first-contributions/issues/107627) for detailed information.
